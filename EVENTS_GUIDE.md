# Сугалаа удирдах заавар

## Events.json файл засварлах

`data/events.json` файлыг засах замаар та сугалааны мэдээллийг удирдана.

### Шинэ сугалаа нэмэх

```json
{
  "events": [
    {
      "id": "event-1",
      "title": "Шинэ жилийн мега сугалаа",
      "startTime": "2026-01-15T20:00:00+08:00",
      "status": "upcoming",
      "prizes": [
        {
          "rank": 1,
          "diamonds": 1000,
          "winners": []
        },
        {
          "rank": 2,
          "diamonds": 500,
          "winners": []
        }
      ]
    }
  ]
}
```

### Талбаруудын тайлбар

- **id**: Сугалааны давтагдашгүй ID (жишээ: "event-1", "event-2")
- **title**: Сугалааны нэр
- **startTime**: Сугалаа эхлэх огноо, цаг (ISO формат)
- **status**: 
  - `"upcoming"` - Ирээдүйн сугалаа
  - `"completed"` - Дууссан сугалаа
  - `"ongoing"` - Явагдаж байгаа сугалаа
- **prizes**: Шагналуудын жагсаалт
  - **rank**: Байр (1, 2, 3...)
  - **diamonds**: MLBB diamond тоо
  - **winners**: Хожигчдын жагсаалт (эхэндээ хоосон)

### Сугалаа дууссаны дараа хожигч нэмэх

```json
{
  "id": "event-1",
  "title": "Шинэ жилийн мега сугалаа",
  "startTime": "2026-01-15T20:00:00+08:00",
  "status": "completed",
  "prizes": [
    {
      "rank": 1,
      "diamonds": 1000,
      "winners": [
        {"name": "Болд"},
        {"name": "Дорж"}
      ]
    },
    {
      "rank": 2,
      "diamonds": 500,
      "winners": [
        {"name": "Батаа"}
      ]
    }
  ]
}
```

**Анхаарах зүйлс:**
1. Сугалаа дууссаны дараа `status`-ийг `"completed"` болго
2. `winners` массивт хожигчдын нэр нэмнэ
3. Олон хожигч байвал массивт олон обект нэмнэ

### Огноо бичих формат

```
"2026-01-15T20:00:00+08:00"
```

- 2026-01-15: Он-сар-өдөр
- T: Цаг эхлэх тэмдэг
- 20:00:00: Цаг:минут:секунд
- +08:00: Цагийн бүс (Монгол цаг)
